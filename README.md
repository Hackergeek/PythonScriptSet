## tinypng
使用tinypng压缩图片
```bash
Usage: tinypng.py [OPTIONS]

Options:
  -f, --file TEXT      单个文件压缩
  -d, --dir TEXT       被压缩的文件夹
  -r, --recursive      是否递归遍历压缩
  -w, --width INTEGER  图片宽度，默认不变
  -o, --overwrite      是否覆盖原文件
  --help               Show this message and exit.
```
